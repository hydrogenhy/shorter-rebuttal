# Run Custom Rules Demo

## Command

```bash
python main.py examples/custom_rules_input.md examples/custom_rules_output.md --custom-rules examples/custom_rules_demo.py
```

## Expected Effects

- `because` -> `鈭礰
- `therefore` -> `鈭碻
- `if and only if` -> `鉄篳
- double spaces in plain text collapse to single spaces
